# 🚀 B2B LeadGen Pro（デモ版）

## 🎯 プロジェクトの概要
「クラウドワークス」等で業務自動化やスクレイピング案件を発注されるクライアント様へ、私の**「高い技術力」**と**「法務リスクを徹底管理するプロ意識」**をお見せするためのデモ用Webアプリケーションです。

企業の公開求人情報を安全に抽出し、B2B向けの営業リスト（CSV）を自動生成するツールを想定して構築されています。

---

## ✨ デモアプリのご案内

**🌐 実際の動作はこちらからお試しいただけます：**  
👉 https://portfolio-scraper.vercel.app

1. **キーワード検索**: 任意のキーワード（例: Python、建設業など）を入力して「検索実行」を押してください。
2. **リアルタイム表示**: バックエンドAPIが動作し、抽出状況がモダンなUIで分かりやすく表示されます。
3. **CSV一括ダウンロード**: 抽出した営業リストをワンクリックでCSV形式でエクスポートできます。

## 🛡️ 本ポートフォリオの最大の強み（あえてデモ仕様にしている理由）

本アプリで「検索実行」を行うと、実在する企業のデータではなく**「ダミーデータ」**が生成されて表示されます。これには、実務を任せていただくプロフェッショナルとしての**「明確な意図とコンプライアンスへの配慮」**があります。

### 1. ⚖️ コンプライアンス第一のリスク管理
既存の著名な求人サイト（Wantedlyやリクナビ等）は、利用規約でスクレイピングを固く禁じているケースがほとんどです。
ポートフォリオの実績作りのためだけにこれらのサイトを無断で標的にすることは、**「コンプライアンス意識の低さ」**の露呈であり、万が一実務で同じことをすれば、**発注者様（クライアント様）に炎上や損害賠償といった甚大な法的リスク**を背負わせてしまいます。

そのため、本デモアプリでは意図的にターゲットを架空のURLに設定し、「ダミーデータを生成して返す安全装置（モック）」を稼働させています。**「絶対にクライアントを危険に晒さない」**という私のお約束の証明です。

### 2. ⚡ 実務への圧倒的な即応性
ダミーデータを返しているのは最後の出力部分のみであり、**システムを支えるコアロジックはすべて本番レベル（プロダクションレディ）で実装済み**です。

* **相手サーバーへの配慮**: 2.0〜3.5秒のランダムなスリープ処理と、不審なアクセスとみなされない適切なUser-Agent設定。
* **耐障害性**: 相手先サイトの通信エラー発生時にシステムをクラッシュさせない堅牢なエラーハンドリング（Vercelのタイムアウト制限も考慮）。
* **UXへの配慮**: 非エンジニアの方でもExcelでそのまま開ける、**絶対に文字化けしないBOM付きのCSVエクスポート機能**。

実務において適法な抽出対象サイトをご指定いただければ、ターゲットURLと数行のコードを差し替えるだけで、この堅牢なシステムを**即座に本番稼働**させることが可能です。
