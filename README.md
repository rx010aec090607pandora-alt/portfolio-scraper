# 🚀 B2B LeadGen Pro（デモアプリ）

## 🎯 プロジェクトの目的
「クラウドワークス」等で案件を発注されるクライアント様へ、私の**「安全への配慮」と「実務での堅牢なシステム構築力」**をお見せするためのデモ用ポートフォリオです。

本ツールは、指定したキーワードに基づいて企業の公開情報（ダミー）を抽出し、**B2B向けの営業リスト（CSV）を自動生成**する機能を持っています。ただ動くツールを作るだけでなく、**「相手先サーバーに迷惑をかけない安全設計」「エラーで止まらない安定性」「非エンジニアにも優しい使い勝手」**を徹底して実装しています。

---

## ✨ 主な機能とデモアプリ

**🌐 デモアプリはこちらからお試しいただけます：**  
👉 https://portfolio-scraper.vercel.app

1. **キーワード検索**: 任意のキーワード（例: ITエンジニア、建設業など）を入力すると、関連する企業リストを抽出します。
2. **リアルタイム表示**: 抽出状況をモダンなUIで分かりやすく表示します。
3. **CSV一括ダウンロード**: 抽出した営業リストをワンクリックでCSV形式でエクスポートできます。

## 🛡️ プロとしてのこだわり（安全設計・リスク管理）
スクレイピング（データ収集）や自動化ツールは、作り方を間違えると相手先のサーバーをダウンさせたり、法的なトラブルに発展するリスクがあります。私は以下の対策を講じており、**「安心して実業務を任せられる設計」**を徹底しています。

* **相手サーバーへの配慮（コンプライアンス）**
  * プログラムがデータを取得する際、人間が操作しているのと同じように**「ランダムな待機時間（2.0〜3.5秒）」**を自動的に設けています。これにより相手先サーバーに過度な負荷をかけることを防ぎます。
  * 一般的なブラウザ（Google Chromeなど）からのアクセスであることを明示する「User-Agent」を適切に設定し、不審なアクセスとして遮断されるリスクを排除しています。
* **絶対に止まらない堅牢なエラー処理（耐障害性）**
  * 相手先サイトの通信エラーや突然の仕様変更でデータが取れなかった場合でも、ツール自体が強制終了・フリーズすることはありません。画面上に適切に「データの取得に失敗しました」というメッセージを出し、被害を最小限に抑えます。
* **文字化けさせない完璧なデータ出力（UXへの配慮）**
  * 出力したCSVをWindowsのExcelで開いた際、**絶対に日本語が文字化けしない**よう、ファイルの先頭にBOM（Byte Order Mark）を付与しています。
  * また、企業名などに「,（カンマ）」が含まれていてもExcel上で表のレイアウトが崩れないよう、すべてのデータを安全に加工（エスケープ処理）してから出力します。
